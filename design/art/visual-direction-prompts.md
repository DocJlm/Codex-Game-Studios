# Visual Direction Prompts

These prompts generated the first Moonlight Dispatch concept assets. Keep them as traceable art direction, not final asset contracts.

## Key Art

Create key art for a 2D top-down route-planning light-stealth game called Moonlight Dispatch. A small courier in a red scarf crosses a crooked moonlit street carrying a sealed parcel, while warm patrol-lamp cones sweep across wet cobblestones. Foggy folklore town, hanging signs, blue moon haze, cozy dark mystery. Strong readable silhouette, premium indie game feel, painterly 2D illustration, no logo text, no watermark, no UI, no gore, no combat. Composition should work as a Steam hero background with room near top for a title later.

Output:

```text
assets/art/concept/moonlight-dispatch-key-art.png
```

## Steam Capsule Concept

Create a Steam capsule composition concept for Moonlight Dispatch, a premium indie PC game. Wide horizontal capsule layout, central small courier silhouette with red scarf and sealed parcel, moonlit blue folklore town behind, warm patrol-lamp cones crossing the foreground, readable at small size, strong negative space for title on the left, premium indie art direction, painterly 2D, cozy-dark mystery, no actual readable text, no watermark, no UI, no combat.

Output:

```text
assets/art/concept/moonlight-dispatch-steam-capsule-concept.png
```

## Character And Tool Sheet

Create a clean concept sheet for Moonlight Dispatch. Include one small moonlight courier with red scarf and parcel satchel, three NPC archetypes (dispatcher woman with bell motif, old locksmith, shrine keeper), and four tool silhouettes (Shade Cloak, Shortcut Key, Quiet Bell, Moonmark). 2D painterly concept art with crisp silhouettes, neutral parchment background, labels as simple icon groupings but no readable text, no watermark. Cozy-dark folklore town style, suitable for guiding production sprites and UI icons.

```text
assets/art/concept/moonlight-dispatch-character-sheet.png
```
