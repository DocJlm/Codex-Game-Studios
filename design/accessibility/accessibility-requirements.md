# Accessibility Requirements

Accessibility tier: Standard

## Required For MVP

- Keyboard remapping or clear documented default controls.
- Gamepad support for movement, interact, pause, and tool use.
- Text size readable at 1080p.
- Timer and exposure warnings must use color plus shape or motion.
- Audio cues must have visual equivalents.
- No rapid flashing.
- Pause available during non-critical moments.

## Localization

- English is source language.
- Simplified Chinese ships at launch if all UI and dialogue strings are centralized.

## Difficulty Support

- Forgiving default exposure meter.
- Optional relaxed timers if playtests show route planning becomes too stressful.
- Fast restart for failed nights.
