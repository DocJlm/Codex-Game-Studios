# Epic: MVP Core Loop

Goal: Deliver the smallest playable Moonlight Dispatch loop that proves movement, job flow, timer pressure, light exposure, tool use, and debrief.

## Success Criteria

- Player can start a night and accept a delivery.
- Player can move through a graybox town route.
- Timer and light exposure can fail the delivery.
- Destination can complete the delivery.
- Debrief explains success or failure.
