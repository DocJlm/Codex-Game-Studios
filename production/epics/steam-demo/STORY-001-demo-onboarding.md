# Story: Demo Onboarding

Status: Ready
Owner role: ui-ux-designer

## Context

Teach the player movement, delivery, light exposure, and one tool without a long tutorial.

## Acceptance Criteria

- First delivery introduces movement and destination.
- Second delivery introduces patrol light.
- Third delivery introduces Shade Cloak.
- Tutorial text can be skipped by returning players.

## Test Plan

- Naive playtest: player completes first delivery without explanation.
- Smoke: tutorial prompts do not block retry flow.
