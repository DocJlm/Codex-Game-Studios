# Story: Shade Cloak Tool

Status: Ready
Owner role: gameplay-programmer

## Context

Implement the first route-altering tool for the vertical slice.

## Acceptance Criteria

- Shade Cloak can be activated during delivery.
- It prevents one short light exposure penalty.
- HUD shows availability and spent state.
- Tool use has clear visual/audio feedback.

## Test Plan

- Manual smoke: activate cloak before entering patrol light.
- Manual smoke: confirm cloak cannot be reused after spent.
