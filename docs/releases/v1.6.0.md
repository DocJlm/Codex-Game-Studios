# Codex Game Studios v1.6.0

v1.6.0 rounds out the public user documentation for Codex Desktop setup, repo-local plugin fallback, upgrades, CI reproduction, and community contributions.

## Changes

- Added `docs/getting-started/quick-start.md` for the copy-paste first-session path.
- Added `docs/install/codex-desktop.md` for Codex Desktop plugin discovery, verification, and fallback behavior.
- Added `docs/install/upgrade.md` for main/tag updates and maintainer release steps.
- Added `docs/platforms/ci.md` for GitHub Actions shape and local reproduction.
- Added `docs/community/contributing.md` for contribution and issue-reporting expectations.
- Added `tools/validate_user_docs.py` and wired it into the full validator suite.
- Bumped plugin metadata to `1.6.0`.

## Validation

Windows:

```powershell
python tools\run_all_validators.py
```

macOS:

```bash
python3 tools/run_all_validators.py
```

Additional focused check:

```bash
python3 tools/validate_user_docs.py
```

## Compatibility Notes

- Skill, role-card, rule, template, and marketplace paths remain unchanged.
- Runtime hooks remain disabled by policy.
- Path-based skill invocation remains supported when Codex Desktop does not expose repo-local plugin discovery.
- Godot remains optional for repository validation and CI.
