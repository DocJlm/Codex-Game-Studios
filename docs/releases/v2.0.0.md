# Codex Game Studios v2.0.0

v2.0.0 is the Final parity gate release. It verifies that Codex Game Studios is a Codex-native complete port of upstream `Donchitos/Claude-Code-Game-Studios` at commit `984023ddac0d5e27624f2baacde6105e45de375f`.

## Changes

- Rechecked upstream HEAD on 2026-05-22 and confirmed it still points to `984023ddac0d5e27624f2baacde6105e45de375f`.
- Confirmed parity counts: 73 skills, 49 role cards, 11 path rules, 40 templates, 12 legacy hook references, and 126 testing-framework Markdown docs/specs.

- Added `tools/validate_complete_port.py` and wired it into `tools/run_all_validators.py`.

## Final Parity Gate

No unexplained parity gaps remain.

- Upstream skills are represented as `cgs-*` Codex skills.
- Upstream agents are represented as role cards.
- Upstream path rules, templates, studio docs, engine references, and testing framework specs are preserved or adapted under repo-local plugin paths.
- Upstream hook intent is represented through reference scripts, explicit validators, skill instructions, CI, and release protocol checks.
- Runtime hooks remain disabled by policy because hidden runtime behavior would make fresh-clone validation inconsistent across Codex installs.

## Compatibility Notes

- Godot remains optional for repository validation and CI.
