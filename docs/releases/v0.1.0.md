# Codex Game Studios v0.1.0

Initial public release of Codex Game Studios, a Codex-adapted version of Claude Code Game Studios.

## Included

- 73 `cgs-*` Codex skills.
- 49 studio role cards.
- 11 path rules.
- 40+ design and production templates.
- Godot, Unity, and Unreal reference material.
- Repo-local Codex plugin manifest and marketplace entry.
- Empty-game smoke fixture for onboarding and story workflow checks.

## Install And Use

Open this repository in Codex and enable the repo-local plugin from `.agents/plugins/marketplace.json` if your Codex build exposes local plugin installation.

Fallback prompt when local plugin install is unavailable:

```text
Use the skill at plugins/codex-game-studios/skills/cgs-start/SKILL.md to set up this project.
```

## Validation

The v0.1.0 tag was validated locally with:

```powershell
python tools\validate_cgs.py
python tools\validate_smoke_fixture.py
python tools\scan_legacy_tokens.py
```

## Known Limits

- Legacy Claude hooks are preserved as reference scripts only and are not wired into Codex runtime hooks.
- Core 14 workflows are Codex-native; the remaining workflows are mechanically migrated and validated structurally.
- Role cards are reference prompts, not automatic subagents.
