# Codex Game Studios v1.0.0

v1.0.0 freezes the core Codex Game Studios public interfaces and adds a CI-backed readiness checklist.

## Highlights

- Added `docs/v1-readiness/freeze-checklist.md`.
- Added `tools/validate_v1_readiness.py`.
- Wired the v1 readiness validator into GitHub Actions.
- Updated README and AGENTS validation checklists.
- Bumped plugin metadata to `1.0.0`.

## Frozen Interfaces

- Existing `cgs-*` skill names remain stable.
- Plugin manifest path remains `plugins/codex-game-studios/.codex-plugin/plugin.json`.
- Marketplace path remains `.agents/plugins/marketplace.json`.
- Role cards remain reference prompts, not automatic subagents.
- Runtime hooks remain intentionally disabled.
- Validation stays explicit through scripts under `tools/`.

## Validation

Run from the repository root:

```powershell
python tools\migrate_from_claude.py
python tools\prepare_v01.py
python tools\validate_cgs.py
python tools\validate_skills.py
python tools\validate_smoke_fixture.py
python tools\validate_transcripts.py
python tools\validate_plugin_install_docs.py
python tools\validate_hook_policy.py
python tools\validate_examples.py
python tools\validate_workflow_polish.py
python tools\validate_v1_readiness.py
python tools\scan_legacy_tokens.py
```

## Compatibility

Future v1.x releases may add docs, examples, validators, templates, role cards, or new skills. Breaking changes to existing skill names, manifest shape, marketplace path, or runtime hook policy require a new major version.
