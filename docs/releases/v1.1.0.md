# Codex Game Studios v1.1.0

v1.1.0 hardens Windows and macOS usability for clone, validation, documentation, and fallback skill usage.

## Highlights

- Added `tools/run_all_validators.py` as the single cross-platform validation entrypoint.
- Added `tools/validate_cross_platform.py` for Windows and macOS documentation and CI shape checks.
- Updated GitHub Actions to run validation on `windows-latest`, `macos-latest`, and `ubuntu-latest`.
- Added `docs/platforms/windows.md`, `docs/platforms/macos.md`, and `docs/getting-started/first-run.md`.
- Removed POSIX-only count checks from CI in favor of Python validators.
- Bumped plugin metadata to `1.1.0`.

## Validation

Windows:

```powershell
python tools\run_all_validators.py
```

macOS:

```bash
python3 tools/run_all_validators.py
```

Release maintainers should also run:

```text
python tools/migrate_from_claude.py
python tools/prepare_v01.py
python tools/run_all_validators.py
```

## Compatibility

- Existing `cgs-*` skill names remain stable.
- The plugin manifest path and marketplace path remain unchanged.
- Runtime hooks remain intentionally disabled.
- Path-based skill invocation remains supported when plugin discovery is unavailable.
