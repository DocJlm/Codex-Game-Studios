# Codex Game Studios v0.4.0

v0.4.0 focuses on local plugin installation UX. Codex builds may expose repo-local plugin installation differently, so this release documents both the preferred marketplace path and the reliable path-based fallback.

## Highlights

- Added `docs/install/local-plugin.md` with the expected marketplace, plugin manifest, and skill paths.
- Documented verification prompts for `$cgs-help`, `$cgs-project-stage-detect`, and starter workflows.
- Documented fallback prompts that call `SKILL.md` files by path when local plugin installation is unavailable.
- Added troubleshooting checks for repo-root selection, JSON parsing, manifest paths, stale discovery, and runtime hook expectations.
- Added `tools/validate_plugin_install_docs.py` and wired it into CI.

## Validation

Run these from the repository root:

```powershell
python tools\migrate_from_claude.py
python tools\prepare_v01.py
python tools\validate_cgs.py
python tools\validate_skills.py
python tools\validate_smoke_fixture.py

python tools\scan_legacy_tokens.py
python -m json.tool plugins\codex-game-studios\.codex-plugin\plugin.json
python -m json.tool .agents\plugins\marketplace.json
```
