# Codex Game Studios v1.2.0

v1.2.0 upgrades Spark Sprint from a static fixture to an optional runnable Godot example while keeping validation usable on Windows and macOS without Godot installed.

## Highlights

- Added `examples/spark-sprint/scenes/main.tscn`.
- Added `examples/spark-sprint/src/gameplay/main_scene.gd`.
- Updated Spark Sprint scripts for movement, spark pickup, HUD updates, timer, win, timeout, and reset.
- Added `tools/validate_godot_example.py`; it reports `SKIP` when Godot is unavailable and loads the scene when Godot is found.
- Added Godot example validation to `tools/run_all_validators.py`.
- Bumped plugin metadata to `1.2.0`.

## Validation

Windows:

```powershell
python tools\run_all_validators.py
python tools\validate_godot_example.py
```

macOS:

```bash
python3 tools/run_all_validators.py
python3 tools/validate_godot_example.py
```

If Godot is not installed, the Godot validator still passes after reporting `SKIP`.

## Compatibility

- Existing `cgs-*` skill names remain stable.
- The plugin manifest path and marketplace path remain unchanged.
- Runtime hooks remain intentionally disabled.
- Godot is optional for repository validation and CI.
