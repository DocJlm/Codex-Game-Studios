# Codex Game Studios v0.9.0

v0.9.0 records current local plugin discovery behavior and keeps the fallback path explicit for users whose Codex build does not expose repo-local plugin installation.

## Highlights

- Added `docs/install/field-test-2026-05-22.md`.
- Documented success signs for repo-local plugin discovery.
- Documented fallback signs for direct `SKILL.md` path invocation.
- Added install problem reporting guidance.
- Extended `tools/validate_plugin_install_docs.py` to cover the field note.
- Bumped plugin metadata to `0.9.0`.

## Validation

Run from the repository root:

```powershell
python tools\migrate_from_claude.py
python tools\prepare_v01.py
python tools\validate_cgs.py

```

## Compatibility

- No skill names changed.
- No marketplace path changed.
- No runtime hooks were added.
- Path-based skill invocation remains supported when plugin discovery is unavailable.
