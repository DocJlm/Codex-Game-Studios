# Codex Game Studios v0.7.0

v0.7.0 polishes the second batch of production workflows used around story readiness, scope control, evidence review, regression coverage, and release readiness.

## Highlights

- Rewrote `$cgs-story-readiness` with clearer readiness rules and next-action routing.
- Rewrote `$cgs-scope-check` as a concise scope-drift review workflow.
- Rewrote `$cgs-test-evidence-review` as an acceptance-criteria-to-evidence review workflow.
- Rewrote `$cgs-regression-suite` around `report`, `audit`, and `update` modes.
- Rewrote `$cgs-release-checklist` around explicit release scope and go/no-go verdicts.
- Added `docs/workflows/production-readiness-workflows.md`.
- Added `tools/validate_workflow_polish.py` and wired it into CI.
