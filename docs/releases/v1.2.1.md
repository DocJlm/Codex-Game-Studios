# Codex Game Studios v1.2.1

v1.2.1 is a patch release for the optional Spark Sprint Godot runtime check. It keeps Godot optional for general validation while making installed-Godot checks fail on runtime script errors.

## Highlights

- Fixed `examples/spark-sprint/src/gameplay/main_scene.gd` so the scene loads in a clean Godot project without requiring `class_name` registration for child scripts.
- Hardened `tools/validate_godot_example.py` to fail when Godot prints `SCRIPT ERROR:` or `ERROR:` even if the process exits successfully.
- Bumped plugin metadata to `1.2.1`.

## Validation

Windows:

```powershell
python tools\run_all_validators.py
python tools\validate_godot_example.py
```

macOS:

```bash
python3 tools/run_all_validators.py
python3 tools/validate_godot_example.py
```

If Godot is not installed, the Godot validator still passes after reporting `SKIP`. If Godot is installed, it loads `examples/spark-sprint/scenes/main.tscn` and fails on runtime script errors.

## Compatibility

- Existing `cgs-*` skill names remain stable.
- The plugin manifest path and marketplace path remain unchanged.
- Runtime hooks remain intentionally disabled.
- Godot remains optional for repository validation and CI.
