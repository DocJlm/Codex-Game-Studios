# Codex Game Studios v0.2.0

v0.2.0 focuses on making the public adaptation repeatable and verifiable. It does not move the existing v0.1.0 tag; all fixes land in this new release line.

## Highlights

- Repeatable migration: `tools/migrate_from_claude.py` now regenerates the plugin manifest with the real repository, homepage, author, and v0.2.0 metadata.
- Self-contained validation: `tools/validate_skills.py` validates all 73 `cgs-*` skills without depending on a local Codex installation.
- CI coverage: GitHub Actions now runs plugin structure checks, skill validation, smoke fixture validation, JSON validation, legacy token scanning, and migrated asset counts.
- Smoke walkthrough: `tests/fixtures/empty-game/WALKTHROUGH.md` documents the expected shape for the onboarding, stage detection, story pickup, and story closure loop.
- Workflow polish: `$cgs-code-review`, `$cgs-qa-plan`, `$cgs-smoke-check`, `$cgs-architecture-review`, and `$cgs-ux-design` were rewritten as shorter Codex-native workflows.

## Validation

Run these from the repository root:

```powershell
python tools\migrate_from_claude.py
python tools\prepare_v01.py
python tools\validate_cgs.py
python tools\validate_skills.py
python tools\validate_smoke_fixture.py
python tools\scan_legacy_tokens.py
python -m json.tool plugins\codex-game-studios\.codex-plugin\plugin.json
python -m json.tool .agents\plugins\marketplace.json
```

## Fresh Clone Check

Clone the public repository into a temporary directory and run the same validation commands. A passing clone proves the release does not depend on local hidden files such as `~/.codex`.
