# Codex Game Studios v0.6.0

v0.6.0 adds a realistic, static example project that can exercise the Codex Game Studios workflow loop without requiring Godot to be installed.

## Highlights

- Added `examples/spark-sprint/`, a Godot 4.3 / GDScript-style static example project.
- Included concept, systems, architecture, control manifest, epic, story, source drafts, test drafts, smoke checklist, and walkthrough.
- Added `docs/examples/spark-sprint.md` with the recommended prompt sequence.
- Added `tools/validate_examples.py` and wired it into GitHub Actions.
- Bumped regenerated plugin metadata to `0.6.0`.

## Validation

Run these from the repository root:

```powershell
python tools\migrate_from_claude.py
python tools\prepare_v01.py
python tools\validate_cgs.py

```

## Roadmap Impact

This release starts the v0.6-v1.0 track by adding the first realistic example project. Follow-up issues track second-batch workflow polish, install UX field notes, and v1 readiness.
